## 🛠 핵심 역량 (Tech Capabilities)

### 1. AI 에이전트 & LLM
* **LLM 연동 및 프롬프트 제어:** OpenAI(ChatGPT), Claude API를 활용한 애플리케이션 아키텍처 설계
* **에이전트 워크플로우:** LangChain, LangGraph를 활용한 멀티 에이전트 구현 및 자가 성찰(Self-reflection) 로직 설계
* **오픈소스 LLM 활용:** Hugging Face 기반의 경량 모델(LLM) 활용 및 백엔드 연동

### 2. 검색 및 데이터 기술 (RAG)
* **Advanced RAG:** 벡터 데이터베이스(Pinecone, pgvector)를 활용한 고성능 유사도 검색 구현
* **데이터 파이프라인:** BeautifulSoup, Selenium을 활용한 크롤링 및 웹 데이터 정제 자동화
* **검색 성능 최적화:** 문서 Chunking 전략 최적화 및 LlamaIndex 기반 효율적 인덱싱

### 3. 클라우드 & DevOps
* **CI/CD 자동화:** GitHub Actions를 이용한 코드 빌드, 테스트, AWS 배포 파이프라인 자동화
* **컨테이너화:** Docker를 활용한 애플리케이션 가상화 및 배포 환경 일치화
* **인프라 관리:** AWS(EC2, S3, RDS)를 활용한 고가용성 웹 서비스 아키텍처 설계
